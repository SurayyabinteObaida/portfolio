PhD Research

Security of LLM-generated code In progress

PhD Thesis · Jinnah University for Women

Investigates whether proactive vulnerability prevention or post-hoc repair more effectively reduces security flaws in AI-generated web backend code. Constructs a novel multi-language benchmark dataset of web backend tasks, generates code across multiple LLMs, and evaluates three mitigation strategies — hint-based prevention, direct repair feedback, and explained repair feedback — using CodeQL static analysis. Extends prior work (SecurityEval, SecCodePLT) with a web-specific, multi-language scope.
